## 📖 About

**1337-Journey** is the full trace of my time in the 1337/42 Piscine — not just a folder of C exercises, but a complete archive of everything the Piscine demanded: subject PDFs, shell scripts, C projects, exam attempts, algorithmic problem solving, and personal notes.

Every exercise here was built under the same conditions the Piscine imposes: strict deadlines, the Norm, peer evaluations, and no shortcuts. This repository is both a **portfolio** and a **logbook** — proof of work, and a reference I can return to.

<br/>

## ✨ Features

| | |
|---|---|
| 📁 **Organized by day** | Every Shell and C module is isolated in its own folder, mirroring the Piscine's structure |

| 🧠 **Problem solving track** | Grademe exercises, algorithm drills, and independent practice, kept separate from graded modules |

## 🧠 1337/42 Philosophy

This Piscine is built on **peer learning**, **project-based evaluation**, and **radical autonomy**. There are no lectures — only subjects, deadlines, and other students. That structure shapes everything in this repository:

- **No copy-paste solutions.** Every file here was written, debugged, and understood individually.
- **The Norm is not optional.** Clean, consistent C code is part of the exercise, not an afterthought.
- **Peer evaluation matters.** Code was reviewed by peers and reviewed peers' code in return — both are reflected in how these projects evolved.
- **Struggle is part of the process.** Notes and commit history reflect real debugging, not a polished final draft.

<br/>
